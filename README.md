# Material Farmacias

Aplicación web para la gestión de material promocional y corporativo para farmacias.

Proyecto Intermodular del Grado Superior de Desarrollo de Aplicaciones Web.

---

## Estructura del repositorio

```
pi-material-farmacias/
├── backend/    # API REST (Node.js + Express + Prisma)
└── frontend/   # Aplicación web (React + Vite)
```

Cada carpeta tiene su propio README con instrucciones detalladas:

- [backend/README.md](backend/README.md)
- [frontend/README.md](frontend/README.md)

---

## Requisitos previos

- Node.js >= 20
- MySQL >= 8
- npm
